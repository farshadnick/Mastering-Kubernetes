```
kubectl apply -f hpa.yml
kubectl get hpa
```
```
--cpu-percent=50: Target average CPU utilization across all pods should be 50%.
--min=1: Minimum number of pod replicas is 1.
--max=10: Maximum number of pod replicas is 10.
```
